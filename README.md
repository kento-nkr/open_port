# OPERN_PORT

WindowsからWSL2のUbuntuに対してポートフォワーディングをし，firewallを無効化するためのpowershellスクリプト

# HOW TO USE

> [!CAUTION]
> This program may expose your computer to security risks.

- open port

  > `./open_port.ps1`

- check status

  > `./open_port.ps1 status`

- clear port settings

  > `./open_port.ps1 clear`
